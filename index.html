How it works

From URL to insight in seconds

No sign-up, no install, no config. DevLens runs entirely in your browser against the public GitHub API.

01

Paste a PR URL

Drop in any public GitHub pull request link. DevLens validates and parses it instantly.

02

Fetch & parse the diff

DevLens pulls files, commits, and patch data through the GitHub API and parses every hunk.

03

Get the full picture

Visual stats, an algorithmic risk score, and a navigable line-by-line diff breakdown.

Features

Everything a reviewer actually needs

Purpose-built for code review. Nothing more, nothing less.

Diff Viewer

Color-coded line-by-line changes with a collapsible file tree. Jump to any hunk in one click.

Risk Score

An algorithmic complexity rating from 0 to 100, weighing churn, file spread, and hot paths.

Commit Timeline

A visual history of every commit in the PR, so you can follow the author's train of thought.

Language Breakdown

A donut chart of file types changed. Spot config-heavy or test-light PRs at a glance.

Keyboard Shortcuts

Press ⌘K to search, J/K to navigate files. Review without touching the mouse.

Dark Mode

Built-in and on by default. Your preference is saved automatically across sessions.
